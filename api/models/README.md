# Yolo Model

## Overview

This is the Yolo Model Api. This repositary is mainly used as a submodule for the yoloApi and the yoloSocket.


## Getting Started

Clone the repo:
```sh
git clone https://github.com/yolotechnologies/yoloModel.git
cd yoloModel
```

Install yarn:
```js
npm install -g yarn
```

Install dependencies:
```sh
yarn
```
