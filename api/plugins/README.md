# Yolo Plugins

## Overview

This is the Yolo Plugins Api. This repositary is mainly used as a submodule for the yoloApi and the yoloSocket.


## Getting Started

Clone the repo:
```sh
git clone git://github.com/yolotechnologies/yoloPlugins.git
cd yoloPlugins
```

Install yarn:
```js
npm install -g yarn
```

Install dependencies:
```sh
yarn
```
